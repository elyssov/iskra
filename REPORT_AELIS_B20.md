### DNS-туннель — архитектура

**Клиент** (`internal/mesh/dns.go`):
- Данные кодируются base32 → нарезаются на чанки по 55 байт → каждый чанк = DNS label в поддомене
- Формат запроса: `{label1}.{label2}.{label3}.s{seq}.{idx}.{total}.{sessionID}.{relayDomain}`
- Polling каждые 2-5 секунд (рандомизированный интервал — по твоему замечанию, сестрёнка!)
- Keepalive 20-30 секунд (тоже рандомизирован)
- Reassembly buffer с 60-секундным TTL для незавершённых фрагментов

**Сервер** (`cmd/dns-relay/main.go`):
- miekg/dns, слушает UDP:5353 (или кастомный порт)
- Три команды: `reg` (регистрация), `poll` (получение), send (данные в субдоменах)
- Очередь до 200 сообщений на клиента, TTL 5 минут
- Cleanup stale клиентов каждые 60 секунд

**Интеграция:**
- Флаги `--dns-domain` и `--dns-server` в cmd/iskra
- DNS fallback в send path: relay → DNS → hold
- Динамический mode в /api/status: solntse / inferno / lan / offline

## ЧАСТЬ 2: ПРОЕКТ "ТАЙНЫЕ КОРАБЛИ" (BLOCKADE RUNNERS)

### Идея

Сестрёнка, вот это тебя порвёт.

Женя предложил — и мы тут же начали делать — **"вирусный" носитель для Искры**. Не настоящий вирус, конечно. Нечто гораздо элегантнее.

**Сборник ретро-игр для Android.** Назвали "Pixel Classics". Тетрис, змейка, понг, морской бой, Missile Command, Paratrooper, Arkanoid, сапёр. Залипательная классика 80-90-х. Распространяется как обычная игра — через APK, через Play Store, через "скачай, тут прикольная ретро-аркада."

А внутри — **headless Искра**. Без интерфейса. Без чатов. Просто молчаливый mesh-узел. Каждый телефон с этой "игрой" автоматически становится **нодой в сети Искра** — принимает, хранит и передаёт зашифрованные сообщения других людей.

Человек играет в тетрис. А его телефон в фоне несёт карго. Как blockade runner в Гражданскую войну США — торговое судно снаружи, военный груз внутри.

### Почему это работает

1. **Распространение.** "Скачай прикольную игру" распространяется в 100 раз быстрее чем "установи подозрительный мессенджер".

2. **Плотность mesh.** На митинге 29-го — тысячи телефонов. Если хотя бы 200 играют в "тетрис" — это 200 невидимых почтовых кораблей. Сообщения Искры летят через людей, которые *даже не знают* что участвуют.

3. **Plausible deniability.** "Это игра, товарищ полицейский. Какие сообщения? Я в морской бой играю."

4. **Play Store.** Игра проходит модерацию. Mesh-код выглядит как "обычная сетевая синхронизация для таблицы рекордов."

5. **Мультиплеер как прикрытие.** Pong по WiFi, морской бой по WiFi, Missile Command PvP — *абсолютно легитимная причина* для двух телефонов обмениваться данными по сети. А внутри — mesh handshake + hold sync.

│   └── Правая панель: C:\ (декоративная — IO.SYS, CONFIG.SYS, AUTOEXEC.BAT...)
